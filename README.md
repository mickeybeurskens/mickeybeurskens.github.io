# Mickey Beurskens Blog ☕️
## About This Blog
Built with:
- [Hugo](https://gohugo.io/) static site generator
- [Paper theme](https://github.com/nanxiaobei/hugo-paper)

Key sections:
- Essays: In-depth articles.
- Projects: Open-source work, talks.

Local setup:
1. Install Hugo
2. Clone repo
3. Run `hugo server`
4. View at http://localhost:1313
